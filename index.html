Raspberry Pi 4 Security Camera

Using OpenCV 4.0 and Pi Camera

This project was really exciting. After purchasing a Raspberry Pi 4 and a Pi Camera module, I quickly tried to install OpenCV to build my secrity camera. Turns out, it was a difficult process for a complete beginner with lacking knowledge in CLI commands.

After quite a bit of effort, I overcame my technical difficulties and I was able to get it set up. After integrating some Python scripts, I was able to get live video footage on a web server, as well as automatic emails with security updates when a person is detected.
